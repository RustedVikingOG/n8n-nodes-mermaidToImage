      - [ ] API routes and validation

```

  │   └── POST /api/auth/register
  ├── Frontend
  │   ├── RegisterForm component

  │   ├── Registration success/error handling
  │   └── Responsive design (mobile/tablet/desktop)
  └── Tests

```
